# Multi-omics-study-of-HCC-gut-microbiome

The gut microbiota is increasingly recognized as a modulator of hepatocellular carcinoma (HCC) prognosis. We performed a gut metagenomic analysis of 1,452 individuals across three cohorts comprising HCC, cirrhosis, hepatitis patients and healthy controls.
